xSACdb, a web application for managing BSAC branch clubs, is an open source project built and maintained by Will Pimblett.

Feature requests, bug reports or general comments can be sent to xsacdb@wjdp.uk . For technical support please contact your club's IT administrator.

As an open source project we rely mainly on volunteers to contribute to the project. If you'd like to help out see the project's page on GitLab. Non-exhaustively we need the help of programmers, designers and writers.

If you'd like to get involved but aren't sure how please email on the address above.
