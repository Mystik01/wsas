Here Be Dragons

The debug backend is a very vanilla Django admin site, it is used to make modifications to the database when the xSACdb site just isn't cutting it. Be wary though!

  • No Error Checking Beyond basic model validation checks there will be no proper error checking, you could very easily modify/delete something you didn't mean to. This could cause the main xSACdb application to crash or to malfunction.
  • Security Changing settings can invalidate the security of this application.
  • Lastly Power, and lots of it Everything can be modified, useful and dangerous.

You should only continue if you know what you're doing, or are being guided by someone who does!
