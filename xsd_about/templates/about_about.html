xSACdb

xSACdb is an open source project built and maintained by Will Pimblett.

As of some point recently the project contains around 37,000 lines of code. If you'd like to contribute please see the project's page on GitHub.

xSACdb is powered by the fabulous Django web framework, alongside additional projects.

Powering xSACdb:

  • Pillow
  • South
  • Unidecode
  • argparse
  • django-appconf
  • django-bootstrap-toolkit
  • django-debug-toolbar
  • django-facebook
  • django-geoposition
  • django-tastypie
  • docutils
  • mimeparse
  • python-dateutil
  • six
  • sqlparse
  • wsgiref
